Link your WooCommerce eshop web site to your Mautic based marketing automation platform

== Description ==

This plug-in is designed to help you link your WooCommerce eshop web site to your Mautic based marketing automation platform. Easily select what customer and order related data you want to import into Mautic once your customer has placed his/her order.

Various exported data help improving marketing tasks:
* Customer related exported data may include first name, last name, email, phone, address.
* Order related exported data may include product ID

== Installation ==

<h4>Plug-in installation</h4>
1. Upload the plugin files to the `/wp-content/plugins/woo-mautic` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Settings->Mautic screen to configure the plugin.
1. Enter your Mautic credentials.

<h4>Plug-in configuration</h4>
1. Once connected to your Mautic setup, you will be able to make a link between your WooCommerce customer information and your Mautic marketing automation.
1. In order to the fields to automatically update in Mautic, please make sure that the concerned fields are marked as "Publicly updatable".

== Frequently Asked Questions ==

= Do I need a running copy of WooCommerce? =

Yes. This plug-in adds Mautic specific functionalities to your existing WooCommerce installation.

= The Mautic fields are not updated. =

Make sure that the concerned fields are marked as "Publicly updatable".
